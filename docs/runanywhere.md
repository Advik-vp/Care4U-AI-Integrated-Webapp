# Run Anywhere

## Prerequisites
- Node.js (v16+)
- Python (v3.8+)
- Git

## Setup Frontend
1. cd frontend
2. npm install
3. npm start

## Setup Backend
1. cd backend
2. pip install -r requirements.txt
3. python main.py

## Environment Variables
- Copy .env.example to .env and fill in API keys for OpenAI, Claude, etc.

## Running the App
- Frontend: http://localhost:3000
- Backend: http://localhost:5000
